An ML-powered parking system that detects vacant spots via camera + segmentation, stores results in PostgreSQL, and guides users to spots through a web UI with path planning.

## Features

- **Camera-based detection** of vacant parking spots using image segmentation
- **PostgreSQL backend** for storing parking lot state
- **Web UI** displaying real-time availability and guiding users to open spots
- **Path planning** to route drivers efficiently to the nearest vacant spot
