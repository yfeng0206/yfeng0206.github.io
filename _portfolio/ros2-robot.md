Prototyped a differential-drive robot stack in ROS 2 Jazzy with URDF modeling, C++/Python nodes, and Gazebo simulation.

[View on GitHub](https://github.com/yfeng0206/ROS2_Intro){: .btn .btn--primary}

## Features

- **URDF robot model** with differential-drive kinematics
- **C++ and Python ROS 2 nodes** for control and sensing
- **Gazebo simulation** with RViz2 visualization
- **Localization stack**: wheel odometry, TF2 transforms, IMU simulation, EKF sensor fusion
- Configured via ROS 2 launch files and parameter configs
